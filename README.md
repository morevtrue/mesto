**Ссылка на проект**

* [Github](https://morevtrue.github.io/mesto/)

**Описание**

* Учебный проект Яндекс.Практикум о красивых местах мира тыглазами других людей. Использованы такие технологии, как БЭМ, flex-box, grid-сетки, медиа-запросы, подключение шрифтов локально, относительные величины измерения, адаптивная верстка, подключение JavaScript для оживления интерфейса.

**Что можно улучшить**

Адаптив, попап, добавить новые попапы, плавность открытия/закрытия попапов.
